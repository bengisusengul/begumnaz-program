# Agent 01 — Fonksiyonel / Bütüncül Tıp Doktoru Perspektifi

> Perspektif: Begümnaz'ın 16 ürünlük protokolünü yazan doktorun düşünce çerçevesi (mizaç temelli, post-iso bariyer, endo yan etki kontrolü).

## 🎯 Bu Perspektifin Vurguları

### 1. Mizaç Temelli Yaklaşım

Doktor klasik allopatik (sentetik ilaç + tek mekanizma) yerine bütüncül yaklaşım uyguluyor:
- **Vücut sıcaklığı** = sağlık temeli (sıcak su + ayak banyosu + termal giyim)
- **Gut microbiome** = sistem desteği (Akavital prebiyotik + Duobalance probiyotik)
- **Karaciğer detoks** = ana axis (Lifextra On Hepa + NAC + Anti turmeric)
- **Mitokondri enerji** = hücresel düzey (NADH + Benfolife B1 + MM-12 B12)
- **Mineral denge** = eksiklik düzeltme (Lugol iyot için tiroid + magnezyum kas/uyku)
- **Emosyonel denge** = beden-zihin (Bach çiçekleri + ıtır aromaterapi)

### 3. Anti-Inflamatuvar Yığın

Begümnaz'da kronik inflam belirteçleri (CRP, ALT) normal AMA endo + post-iso cilt + hormonal akne **lokal inflam** kaynakları. Doktor sistemik anti-inflam yığını:

- **Anti turmeric** (curcumin) → NF-kB inhibisyon
- **NAC** (N-acetylcysteine) → glutatyon prekursor + antioksidan + lung+karaciğer
- **More than** (MSM + Vit C) → eklem anti-inflam + antioksidan
- **Bach çiçekleri** → emosyonel denge → kortizol denge → inflam dolaylı
- **Itır yağı** (haricen) → aromaterapi + sirkülasyon

### 4. Gut-Skin-Hormone Axis

Doktor bu üç ekseni birlikte hedefliyor:

**Gut**:
- Akavital prebiyotik (akasya gam fiber)
- Duobalance probiyotik (gece, gut microbiome)
- Gül sirkesi (mide asit + sindirim destek)
- Alkemist oil pulling (ağız florası — gut microbiome başlangıç)

**Skin**: Sistemik destek (akneye dolaylı) + lokalize Acnelyse + Madescar

**Hormone**:
- Lugol iyot (tiroid axis)
- Time Magnezyum (HPA axis + uyku)
- Vit D (Colewinde) (endokrin destek)

### 5. Mizaç + Modern Bilim Sentezi

Doktor klasik modern bilim referansları + mizaç tıbbı geleneksel bilgi karışımı:
- **Modern**: Lugol iyot dozaj (UVA Iowa tiroid çalışmaları), magnezyum sülfat banyo (cilt yolu emilim çalışmaları), NAC glutatyon (lung + karaciğer literatürü)
- **Geleneksel**: Mizaç temelli diyet (sıcak/soğuk denge), ıtır yağı (Pelargonium graveolens aromaterapi), gül + zeytin yaprağı + aynı safa çayları (Anadolu bitki tıbbı), oil pulling (Ayurvedic kökenli, ağız florası)

### 6. Bu Perspektifin SYNTHESIS'e Katkısı

- **Mizaç temelli yaklaşımın bütünlüğü** — sadece ilaç değil, yaşam tarzı + diyet + aromaterapi + bedensel pratikler (ayak banyosu) hep birlikte
- **Doktor protokolü dışına çıkma kuralı** — doktor zaten görerek karar verdi, ek takviye eklenmez
- **Vücut sıcaklığı = sağlık temeli** felsefesi
- **Gut + skin + hormone axis** ortak hedef
