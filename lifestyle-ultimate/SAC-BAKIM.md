# Saç Bakımı — Yağlı Kafa + Kuru Uç Paradoksu

> **Saç tipi**: Orta yoğunluk + seyrek + dalgalı + kuru uçlar + **çok hızlı yağlanan kafa derisi**.

## 🎯 3 Çekirdek Prensip

1. **Kafa derisini KURUTMA** — aşırı sülfatlı şampuan veya çok sık yıkama sebumu artırır (overcompensation). Sülfatsız + dengeleyici şampuan.
2. **Uçları doyur** — yağ, mask, leave-in. Kafa derisine gelmesin.
3. **Az ama akıllı yıka** — hızlı yağlanmaya cevap "her gün yıkamak" değil; **kuru şampuan + scalp care** ile yıkama aralığını uzat (hedef: haftada 3 yıkama).

## 🧖 Yıkama Tekniği (Yağ-Kuru Paradoksu İçin)

1. **Yıkamadan önce** → uçlarına argan/badem yağı sür (1-2 yk)
2. **Şampuan sadece kafa derisine** (parmak uçlarıyla masaj 1-2 dk, **tırnak YOK**) → uçlara sabunu akıttığın suyla yetersin
3. **Soğukca durulama** son aşamada (kapakçık kapanır, parlaklık artar)
4. **Saç krem/conditioner sadece uçlara** (kafa derisinden 1 karış aşağı)
5. **Havlu sürtme YOK** → pamuk t-shirt ile sıkma
6. **Hava ile kurutma** ideal. Mecbur fön → soğuk modda, kökten uca

## 🌿 İçten Destek (Doktor Protokolü Üzerinden)

Saç sağlığını destekleyen mevcut protokol ürünleri zaten var — **ekstra takviye gerekmez**:

- **NBT Life Guard** (multivit benzeri) — saç/tırnak destek için temel mikrobesinler
- **Anti MM-12** — doktorun verdiği vitamin desteği
- **Anti turmeric** — antiinflamatuvar (cilt + saç)
- **NAC** — antioksidan + saç bütünlüğü için faydalı (glutatyon)
- **Colewinde Vit D** — saç follikülü sağlığı
- **Time Magnezyum** — stres + uyku → saç dökülmesi azaltır
- **Diyet protein**: Kuzu/koyun/hindi 3x/hafta + yumurta + çiğ badem/fıstık/kabak çekirdeği = aminoasit + çinko + magnezyum doğal kaynaklı

**Önemli**: Kan testinde **serum demir YÜKSEK** (204) → demir hapı / B12 takviyesi / saç dökülmesi takviyeleri **kendi kafamıza eklenmez**. Doktor paneli görerek protokolü düzenledi.

## 🎯 Hedefler

- **Uzun saç** — gelişim için sağlıklı kafa derisi + uç koruması
- **Güçlü saç** — protein (diyet) + dış bakım

- **Hızlı yağlanma azaltma** — sülfatsız şampuan + scalp tonic + az yıkama
