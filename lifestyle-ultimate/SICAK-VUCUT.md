# Vücut Sıcaklığı Yönetimi — Sıcak Bölgeler

> Doktor talimatı: **Özel bölge + karın + popo + bel + ayaklar** mutlaka sıcak tutulacak. Mizaç tıbbında bu bölgeler kadın üreme sağlığı + endo için kritik.

## 🌡 5 Sıcak Tutulacak Bölge

1. **Özel bölge** (pelvik / üreme sistemi)
2. **Karın** (alt karın özellikle)
3. **Popo** (kuyruk sokumu + kalça)
4. **Bel** (lomber)
5. **Ayaklar**

Bu bölgelerden herhangi biri soğursa endo flare tetiklenebilir; pelvik ağrı + bloating artar.

## 👚 Giyim Pratiği

### Yıl Boyu (Kış Dahil)

| Ne | Niye |
|---|---|
| **Termal/yünlü iç çamaşırı** (yüksek bel) | Bel + alt karın hizası kapalı |
| **Yüksek bel pantolon/eşofman/kemerli pijama** | Bel açıkta kalmasın |
| **Yün veya kalın pamuk çorap** ev içinde sürekli | Ayaklar soğumasın |
| **Termal alt** (sıcak coğrafyada bile) | Karın + popo kapalı |

### Yaz Aylarında

- **Klima**: Doğrudan vücuda üflemesin; alt karın hizasına soğuk hava değmesin
- **Cool keten/pamuk** yüksek bel kıyafet (tişört içe sokulu kalsın)
- **Plaj/havuz**: Islak mayodan hemen çıkıp kuru kıyafete geç; soğuk havuz sonrası bel + karın sıcak tutmak için çıkışta kurutma + havlu

### Yatak

- **Yorgan kalın** (vücudu sıcak tut)
- **Yatak odası serin** (rahat uyku) — ama vücut katmanlarla sıcak
- **Yün veya termal çorap** uykuda

## 🚫 Yapılmaması Gerekenler

| ❌ Yapma | Niye |
|---|---|
| **Çıplak ayak yer** | Ayaklar yer üzerinde soğur |
| **Soğuk yere oturma** | Mermer, taş, soğuk yer → pelvik bölge soğutma |
| **Düşük bel pantolon** | Bel açıkta kalır |
| **Crop top tek başına** (kış) | Alt karın açık |
| **Islak saçla uyuma** | Boyun + sırt soğutma |
| **Klima doğrudan vücuda** | Direkt üfleme = lokalize soğutma |
| **Soğuk yatak** | Vücut ısısı geceleyin düşer, üşüme ile uyku bozulur |

## ✅ Pratik İpuçları

### Sabah Hazırlık

1. Uyandıktan hemen önce **çorap giy** (yataktan çıplak ayak çıkma)
2. Pijama üstüne **sweat/hırka** (özellikle kış)
3. Kahvaltıya geçişte termos sıcak su hazır

### Gün İçi

- Ev içinde **terlik + kalın çorap** birlikte
- Dışarı çıkışta yüksek bel pantolon + iç içe katmanlar
- Soğuk hava → boyun atkısı bel hizasına bağlama (alt karın koruma)
- Eğer otururken üşüyorsa → battaniye/şal bel/karın bölgesine

### Akşam

- Ayak banyosu sonrası (20:30) → kuru havlu + hemen çorap
- Pijama + termal alt (gerekirse iki katman)
- 22:00 yatağa: vücut sıcak → yorgan altına gir

## 🩸 Period + Endo Flare Dönemi

Bu dönemlerde sıcak tutma daha kritik:
- **Heating pad** bele + alt karına (20 dk × 2-3 kez/gün)
- **Sıcak duş** (vücudu ısıtmak için) — ama saçı ıslak bırakma
- **Ekstra ayak banyosu** (sabah ayrıca)
- **Sıcak içecek** sürekli (gül+zeytin yaprağı, aynı safa, papatya)
- Klima/soğuk hava direkt yasak

## 🌬 Hava Şartları

| Hava | Önlem |
|---|---|
| **Soğuk** (5°C altı) | Termal alt + ceket + bere + bel destekli üst (yüksek bel kazak) |
| **Rüzgarlı** | Atkı bel hizasına bağlama opsiyonu |
| **Yağmurlu** | Su geçirmez ayakkabı (ayaklar ıslanmasın); ıslanırsa eve girer girmez ayak banyosu |
| **Klima ortam** (yaz) | Hırka/şal hazır bulundur, kalkıp dolaşırken klimadan uzaklaş |
| **Açık havada** | Yün/pamuk kat kat, çıkarılabilir |

## 💧 Su Tüketimi (Sıcak)

- **Soğuk su YASAK** (doktor protokolü) — buz dolaptan çıkar çıkmaz içme
- **Ilık-sıcak su** günde 5+ bardak (cam termos)
- Plastik şişe yok
- Gün boyu sürekli yudumla (büyük yudumlar değil)

## 🛒 Alışveriş Hatırlatma

```

☐ Yüksek bel iç bel destekli kazak (yumuşak)
☐ Kalın yorgan
☐ Cam termos (sıcak su için)

```
