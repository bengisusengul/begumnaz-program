# Agent 02 — Endometriozis Anti-İnflamatuvar Diyet Uzmanı

> Perspektif: Endo'ya özel anti-inflamatuvar beslenme çerçevesi (klinik diyetisyen).

## 🎯 Bu Perspektifin Vurguları

### 1. Endo + İnflamasyon

Endometriozis kronik inflamatuvar hastalık. Diyet inflamasyonu modüle eder.

**Pro-inflamatuvar yiyecekler** (endo'da yasak):
- Rafine karbonhidrat (beyaz un, beyaz pirinç, şeker)
- Trans yağlar (margarin, hazır gıdalar)
- Omega 6 yüksek yağlar (mısır, ayçiçek)
- Süt ürünleri (özellikle endüstriyel)
- Kırmızı et (özellikle endüstriyel grain-fed)
- Alkol, kahve aşırı
- Glüten (bazı kişilerde)

**Anti-inflamatuvar yiyecekler** (endo'da iyi):
- Sebze (özellikle yeşil yapraklı)
- Meyveler (özellikle berry — burada Begümnaz'ın "ekşi yasak" kuralı çevriliyor)
- Omega 3 (balık, ceviz, keten tohumu)
- Anti-inflam baharatlar (zerdeçal, zencefil, sarımsak)
- Sızma zeytinyağı
- Yeşil çay (ama Begümnaz'da yasak — bitkisel çaylar yerine)

**Begümnaz'ın diyeti** bu prensipte ✓ — sadece tatlı meyve (anti-inflam berry tartışmalı ama doktor "tatlı" demiş, lokal mizaç adaptasyon).

### 2. Endo'ya Özel "Yüksek Risk" Gıdalar

Klinik literatürde endo flare ile bağlantılı:

| Gıda | Mekanizma | Yasak Listede? |
|---|---|---|
| **Süt ürünleri** | IGF-1, östrogen, prolaktin | ✅ Yasak |
| **Kırmızı et fazla** | Saturated fat, östrogen | ⚠️ Sınırlı (haftada 3x kuzu/koyun) |
| **Trans yağ** | Inflamasyon | ✅ Yasak (hazır gıda) |
| **Rafine karb** | İnsülin spike → östrogen | ✅ Yasak (beyaz un, pirinç) |
| **Alkol** | Östrogen metabolizma | (Begümnaz'da bilinmiyor; mizaç doktoru muhtemelen yasak) |
| **Kafein fazla** | Östrogen, kortizol | ✅ Yasak (çay, kahve) |
| **Soya** | Fitoöstrogen tartışmalı | (Yasak listede yok; doktor önerirse atla) |

### 3. Endo İçin Omega 3 (Ek Düşünce)

**Endo'da omega 3 anti-inflam**: Balık yağı + keten tohumu.
- Keten tohumu Begümnaz'ın kahvaltısında var ✓
- Balık (büyük dip balık yasak ama küçük balıklar?) — doktor protokolünde net değil
- **Doktor protokolünde omega 3 takviyesi YOK** → ek ekleme yapılmaz (kullanıcı kuralı)

Yine de **kuruyemiş omega 3**: Çiğ ceviz, çiğ badem, keten = doğal omega 3 kaynakları → Begümnaz'da var.

### 4. Endo İçin Magnezyum (Ek Düşünce)

Magnezyum **endo ağrı azaltıcı** (PMS + dismenore için klinik kanıt).
- Time Magnezyum 2x1 protokolde ✓
- Epsom tuzu ayak banyosu = cilt yolu Mg ✓
- Doğal Mg kaynakları: Yeşil yapraklı sebze (ıspanak, lahana), kuruyemiş (badem, kabak çekirdeği) → Begümnaz'da var

### 5. Endo İçin Lif

Lif → östrogen metabolitlerinin atılımı (önemli, östrogen detoks):
- Sebze pişmiş → yüksek lif
- Çiğ badem, kabak çekirdeği → lif
- Mayasız ekmek (ancient grain) → orta lif
- **Akavital prebiyotik** → ekstra lif (akasya gam) → gut floraya yem ✓

Hedef: Günde **25-35 g lif** (genel yetişkin kadın). Mevcut diyet bunu sağlıyor.

### 6. Insülin + Endo

İnsülin direnci endo'da yüksek (literatür). Begümnaz **insülin direnci yok** (HbA1c %5.10 ✓). AMA önleyici:
- Rafine karb yasak ✓
- Yüksek glisemik yasak ✓
- Düzenli öğün (acıkmadan yemek esnek ama düzen var)

### 7. Bu Perspektifin SYNTHESIS'e Katkısı

- **Endo + inflamasyon = diyet doğrudan müdahale** alanı
- **Süt ürünleri yasağı** endo için kritik (cilt için bonus)
- **Kırmızı et orta** — kuzu/koyun haftada 3x doğru denge
- **Omega 3** doğal kaynaklarla zaten yeterli (ek takviye yok)
- **Magnezyum** protokolde + ek doğal kaynaklarla
- **Lif** günde 25-35 g hedef
- **İnsülin denge** koruyucu
