# Agent 03 — Fonksiyonel Tıp Diyet Uzmanı

> Perspektif: IFM (Institute for Functional Medicine) çerçevesinde Begümnaz'ın diyetinin değerlendirilmesi.

## 🎯 Bu Perspektifin Vurguları

### 1. Fonksiyonel Tıp 5R Yaklaşımı (Gut Healing)

Fonksiyonel tıp gut iyileştirme 5R:

1. **Remove** — İrritanlar kaldır (gluten, süt, şeker, alkol, kafein) — Begümnaz protokolünde ✓
2. **Replace** — Sindirim destek (asit, enzim, safra) — gül sirkesi (HCl benzeri) + Lifextra (karaciğer/safra)
3. **Reinoculate** — Probiyotik + prebiyotik — Akavital + Duobalance ✓
4. **Repair** — Gut mukoza onarım (L-glutamine, çinko, omega 3) — protokolde direkt yok ama multivit + diyet protein dolaylı
5. **Rebalance** — Stres yönetim + uyku + yaşam — yaşam protokolü ✓

Begümnaz'ın protokolü **4/5 R**'yi karşılıyor. Repair için spesifik supplement yok ama dolaylı.

### 2. Yemek Sıralaması (Food Sequencing)

Sindirim için optimal sıra:
1. **Lif** önce (salata pişmiş, sebze çorbası)
2. **Protein** sonra (yumurta, et)
3. **Karbonhidrat** en son (mayasız ekmek, pirinç)

Bu sıra **glikoz spike azaltır** + **doyma artırır** + **sindirim destekler**.

Begümnaz'ın kahvaltı/öğle yemeği bu doğal sıraya uygun.

### 3. Mealtime Spacing

Fonksiyonel tıp **time-restricted eating** öneri:
- 12-14 saat **gecelik açlık** (örn. 22:00 son yemek → 10:00 kahvaltı = 12 saat)
- Daha uzun (16:8) opsiyonel ama herkes için değil

Begümnaz'ın programı:
- 22:00 probiyotik (son alım) → 08:00 uyanış + 08:30 sabah AC = ~10 saat
- Kahvaltı acıkma sinyaline göre (~09:00-11:00) → muhtemelen 11-13 saat açlık
- Doğal time-restricted eating uygulanıyor ✓

### 4. Gut-Brain Axis

Fonksiyonel tıp gut-brain bağlantısı:
- Probiyotik → seratonin sentezi (gut'ta %90 seratonin)
- Prebiyotik → SCFA (short chain fatty acids) → BDNF → mood
- Mg + magnezyum → GABA → uyku
- Itır aromaterapi → parasempatik → uyku + rahatlama

Begümnaz'ın protokolü bu axisi hedefliyor.

### 5. Detoks Pathways

Karaciğer 2 faz detoks:
- **Faz 1** (P450 enzimleri) — hidroksilasyon → ara metabolit
- **Faz 2** (konjugasyon) — glutatyon, glisin, sülfasyon vs.

Faz 1 ile Faz 2 dengeli olmalı; Faz 2 yetersizse ara metabolit birikim (kanser riski).

Begümnaz'da:
- **NAC** (glutatyon prekursor) → Faz 2 destek ✓
- **Anti turmeric** (curcumin) → Faz 2 destek ✓
- **Lifextra On Hepa** → karaciğer destek ✓
- **Sebze (kruciferöz)** lahana, brokoli → Faz 1+2 modüle eder ✓

Mevcut protokol **karaciğer detoks tam destek**.

### 6. Methylation Pathway

Methylation = DNA + nörotransmitter + detoks için kritik biokimyasal süreç.

Methylation bozulursa: yorgunluk, depresyon, hormonal sorunlar.

Anahtar besinler:
- **B12** (metilcobalamin) — Anti MM-12 muhtemelen bu ✓
- **Folat** (metilfolat) — kan testinde 9.70 normal ✓
- **B6** — multivit içerikli olabilir
- **Choline** — yumurta + et = doğal kaynak ✓
- **Magnezyum** — Time Mg ✓

Begümnaz'ın B12 kan testinde **YÜKSEK (738)** → muhtemelen MM-12 zaten methylation destekliyor. Doktor protokolde tutuyor.

### 7. Mineral Status

Kan testindeki:
- **İyot 27** (düşük) → Lugol ✓
- **Selenyum 58** (düşük) → doktor diyetten + multivitten destekliyor (Brazil cevizi gibi ek kaynak doktor önermedi)
- **Çinko 88** (orta) → multivit + kuruyemiş ✓
- **Magnezyum 2.0** (normal) → Time Mg + epsom + kuruyemiş ✓
- **Demir 204** (yüksek) → takviye YOK doğru

### 8. Bu Perspektifin SYNTHESIS'e Katkısı

- **5R gut healing** çerçevesi (Begümnaz 4/5 ✓)
- **Time-restricted eating** doğal uygulanıyor (12+ saat)
- **Karaciğer detoks tam destek** (NAC + Anti turmeric + Lifextra)
- **Methylation pathway** B-vitamin + Mg + protein ile destekli
- **Mineral status** kan değerleriyle uyumlu
