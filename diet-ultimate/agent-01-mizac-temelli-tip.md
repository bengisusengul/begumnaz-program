# Agent 01 — Mizaç Temelli Tıp (Tıbb-ı Nebevi / Unani)

> Perspektif: Doktorun mizaç temelli yaklaşımının arkaplanı + geleneksel Anadolu/İslami tıp temellendirmesi.

## 🎯 Bu Perspektifin Vurguları

### 1. Mizaç Tıbbı Temelleri

Mizaç tıbbı (Tıbb-ı Nebevi, Unani, Galen tıbbı uzantısı) 4 temel sıvı (humor) dengesi:
- **Kan** (sıcak + nemli)
- **Safra** (sıcak + kuru)
- **Balgam** (soğuk + nemli)
- **Sevda / kara safra** (soğuk + kuru)

Yaş kadın endometriozis genellikle **soğuk + nemli** mizaç → balgam ağırlıklı denge bozukluğu. Çözüm: sıcaklık ekleyerek dengelemek.

### 2. "Sıcak Vücut" Felsefesi

Doktor talimatı sıcak su + sıcak yiyecek + sıcak bölgeler tutma:
- **Sıcaklık** = enerji + sirkülasyon + organ fonksiyonu
- **Soğukluk** = stagnasyon + ağrı + immün düşüş

Pelvik bölge (özel bölge + karın + popo + bel) soğursa:
- Kadın üreme sistemi yavaşlar
- Endometriozis flare tetiklenir (geleneksel inanış)
- Kronik pelvik ağrı

Bilimsel parallel: **Visceral hypothermia** (soğuk organlar) → kan akışı azalır → onarım yavaşlar. Sıcak duş + sıcak su → vazodilation → kan akışı artar → onarım hızlanır. Modern integrative medicine de bunu destekler.

### 3. "Acıkmadan Yemek YOK" Felsefesi

Doktorun kahvaltıya sabit saat vermeme nedeni: **Acıkma sinyali = vücudun hazır olduğu sinyal**.

Mizaç tıbbında:
- Mide boş + acıkma = sindirim hazır
- Zorla yemek (acıkmadan) = mide yorgunluk + sindirim bozukluğu

Modern parallel: **Hunger hormone leptin-ghrelin döngüsü** + **circadian eating** (vücudun ritmine uyum) — modern fonksiyonel tıp da benzer öneri.

### 4. Çiğ Sebze YASAK Mantığı

Mizaç tıbbı: Çiğ sebze **soğuk + kuru** → sindirim için zor.
- Pişirme = sebzeyi vücuda kabul edilebilir hale getirir
- Pişmiş sebze sindirim için kolay
- Pişme yöntemi → buhar veya az su sote (besin değer korunur)

Modern parallel: Bazı sebzelerde **anti-nutrient** (oksalat, fitat) pişirme ile azalır. Çiğ kruciferöz sebzeler tiroid için olumsuz (goitrogen) — Begümnaz'ın iyot eksikliği + T3 hafif yüksek var, pişmiş sebze burada daha iyi seçim.

### 5. Süt Ürünleri YASAK

Mizaç tıbbı: Süt ürünleri balgam (mukus) artırır.

Modern parallel:
- **Süt + akne** korelasyon (IGF-1, hormon)
- **Endo + süt** korelasyon (östrogen + inflam)
- **Lakto intolerans** popülasyonun %65+ (Türkiye'de yaygın)
- **Kazein + glüten cross-reactivity** otoimmün hastalıklarda

Begümnaz'ın hormonal akne + endo profili → süt yasağı **doğru karar**.

### 6. Dana + Tavuk YASAK, Kuzu/Koyun/Hindi OK

Mizaç tıbbı:
- **Tavuk balgam artırır** (özellikle endüstriyel)
- **Dana ağır gıda** (sindirim zor)
- **Kuzu/koyun/hindi** sıcak doğa → mizaca uygun

Modern parallel:
- Endüstriyel tavuk antibiyotik + arsenik + omega 6 yüksek (inflam)
- Endüstriyel dana grain-fed → omega 6 yüksek
- Otlak kuzu/koyun → omega 3 ratio iyi (anti-inflam)
- Hindi proteinli + düşük yağ

**Tabii ki**: Endüstriyel kuzu/koyun da olabilir. İdeal: **otlak / organic** kuzu kasaptan.

### 7. Bitkisel Çaylar Sadece İzinli (Çay + Kahve YASAK)

Mizaç tıbbı:
- **Siyah çay** taze yapraktan = balgam çıkarır AMA siyah çay tannins → mineral emilim engeller (Begümnaz demir yüksek, sorun değil ama prensip)
- **Kahve** asit + uyarıcı = soğuk mizaca uygun değil (kortizol artışı + endo flare riski)

**İzinli 3 çay**:
- **Gül + zeytin yaprağı çayı** — kalp sağlığı + immün + tatlı
- **Aynı safa çayı** — geleneksel anti-stres + sindirim
- **Melissa + papatya çayı** — sinir sistemi + uyku

Modern parallel: Bu çaylar **kafeinsiz + anti-inflam fitokimyasallar** taşır. Doktor seçimi mantıklı.

### 8. Bu Perspektifin SYNTHESIS'e Katkısı

- **Sıcak vücut felsefesi** (yaşam protokolü temel)
- **Acıkmadan yeme yasağı** (esnek kahvaltı saati)
- **Çiğ sebze yasağı** (pişmiş)
- **Süt ürünleri yasağı** (cilt + endo destek)
- **Et seçimi** (kuzu/koyun/hindi)
- **Bitkisel çaylar** (3 izinli)
