# Haftalık Cilt Çizelgesi — Aktif Rotasyon

> Gym günü sabit olmadığı için **esnek tret kuralı** uygulanıyor.

## 📅 7 Günlük Şema

| Gün | Aktivite | Sabah | Akşam |
|---|---|---|---|
| **Pzt** | 💪 AĞIRLIK 1 | Su yıka → Hyaluron → **Niacinamide** → Garnier nem → SPF | Temizlik → Hyaluron → **Azelaic** → Madescar |
| **Sal** | 🌸 PELVİK TABAN 1 | Su yıka → Hyaluron → **Vit C** → Garnier nem → SPF | **TRETİNOİN gecesi (sandwich)** ✓ |
| **Çar** | 🧘 PİLATES 18:00 | Su yıka → Hyaluron → **Niacinamide** → Garnier nem → SPF | Temizlik → Hyaluron → Madescar (**bariyer gecesi** — pilates sonrası) |
| **Per** | 💪 AĞIRLIK 2 | Su yıka → Hyaluron → **Vit C** → Garnier nem → SPF | Temizlik → Hyaluron → **AHA** → Madescar (haftada 1) |
| **Cum** | 🧘 PİLATES 18:00 | Su yıka → Hyaluron → **Niacinamide** → Garnier nem → SPF | Temizlik → Hyaluron → Madescar (**bariyer gecesi** — pilates sonrası) |
| **Cmt** | 💪 AĞIRLIK 3 | Su yıka → Hyaluron → **Vit C** → Garnier nem → SPF | Temizlik → Hyaluron → **Azelaic** → Madescar |
| **Paz** | 🌸 PELVİK TABAN 2 | Su yıka → Hyaluron → **Niacinamide** → Garnier nem → SPF | **TRETİNOİN gecesi (sandwich)** ✓ |

**Hafta özeti**: 2 tret + 1 AHA + 2 azelaic + 2 bariyer = 7 farklı akşam, hiçbir aktif birbiriyle çakışmıyor.

## 🎯 Niçin Bu Konumlama?

**Tret günleri = Sal + Paz** (düşük aktivite akşamları):
- Sal: pelvik taban fizyopilates yapıldı sabah/öğle, akşam serbest → tret OK
- Paz: pelvik taban restoratif yapıldı sabah, akşam serbest → tret OK

**Bariyer geceleri = Çar + Cum** (pilates dersi akşamları):
- 18:00-19:30 pilates dersi → yoğun terleme + duş → tret YASAK
- Bariyer onarımı: sadece nemlendirici + Madescar

**Azelaic = Pzt + Cmt** (ağırlık günleri):
- Sabah/öğle ağırlık → akşam normalde sakin → azelaic + Madescar OK
- Eğer akşam ekstra antrenman olursa azelaic'i bir sonraki sakin akşama kaydır

**AHA = Per** (üst vücut ağırlık günü):
- Alt vücut terlemesi az → akşam AHA OK
- Sonraki gün (Cuma) bariyer günü → AHA sonrası recovery uyumlu

## ⚠️ Esnek Tret Kuralı (Gym sabit gün yok)

1. **Default**: Sal + Paz akşamları tret.
2. **Çar + Cum 18:00 pilates akşamlarında tret YASAK**. Bariyer günü.
3. **Eğer Begümnaz başka akşam ekstra gym/spor yaptıysa** → o akşam tret SKIP, bir sonraki sakin akşama kaydır.
4. **Yoğun terleme + duş sonrası en az 1 saat bekle** → cilt damarları kapansın, bariyer toparlasın → sonra tret.
5. **Genel pratik**: Tret uygularken cilt **soğuk + kuru + temiz** olmalı.

## 🔄 İleride Tret 3x/Hafta'ya Çıkış

6-8 hafta toleransa göre değerlendirilir:
- 3. tret günü → Per akşamına alınır
- AHA → Pazar'a kayar
- Yeni şema: Sal + Per + Paz tret; Cmt veya Pzt AHA

## 🚨 Aktif Sivilce Müdahalesi

- **DOKUNMA, SIKMA**. Madescar üzerine ince. PIE/PIH'in ana sebebi sıkmak.
- Aktif iri papüle → spot benzoil peroksit 2.5% (PanOxyl varsa) ya da kil maskesi (haftada 1, sadece T-bölge). İlk etap için zorunlu değil.
- Eğer pus akıyor → temiz pamukla kuruda tutmak, baskı yok.
