# Leke Profili Stratejisi — PIE / PIH / Güneş Lekeleri

Begümnaz'ın cildinde 4 farklı leke türü var. Tedavileri ve süreler farklı.

## 🩹 1. PIE (Post-Inflammatory Erythema) — Kırmızı/Pembe İzler

**Baskın leke tipi**. Geçmiş akne sonrası kalan kırmızı/pembe yamalar — kapanmış damarlar.

Foto kanıt: [`../medical-documents/cilt-fotograflari/PHOTO-2026-05-09-14-33-25.jpg`](../medical-documents/cilt-fotograflari/PHOTO-2026-05-09-14-33-25.jpg) (yanak)

### Tedavi

| Yöntem | Niye işe yarar |
|---|---|
| **Niacinamide** (sabah) | Anti-inflamatuvar, kapiller iyileşmesi |
| **Azelaic acid 10%** (gece, tret olmayan) | Çift fonksiyon: akne + PIE |
| **Centella (Madescar)** | Kızarıklık azaltır, bariyer destek |
| **SPF 50** her sabah, mecbur | UV PIE'yi koyulaştırır + uzatır |
| **Zaman** | 2-3 ay sabırla solar (kendiliğinden) |

### Yapılmaması Gerekenler

- ❌ Sıkma, tırmalama (yeni inflamasyon = yeni PIE)
- ❌ Sert eksfoliyan (PIE alanını uyarır)
- ❌ Çok sıcak su (vazodilation = kızarıklık artışı)

### Beklenti

**Aktif tedaviyle**: 2-3 ay solma. Tamamen kaybolma 6 ay+.

## 🟤 2. PIH (Post-Inflammatory Hyperpigmentation) — Kahverengi Noktalar

**Az sayıda**. Geçmiş akne sonrası kalan kahverengi noktalar — melanin birikimi.

### Tedavi

| Yöntem | Niye işe yarar |
|---|---|
| **Tretinoin (Acnelyse)** Sal + Paz | Hücre yenileme, melanin dağılımı |
| **ISANA Vit C 10%** (sabah) | Antioksidan, melanin sentezi engelleme |
| **Azelaic acid 10%** (gece) | Tirozinaz inhibisyonu |
| **SPF 50** | UV = melanin tetik |

### Beklenti

3-6 ay solma. Tret toleransı gelirse 3x/hafta'ya çıkış hızlandırır.

## 🌞 3. Güneş Lekeleri (Yanaklarda Çil Benzeri)

Yanaklarda yoğun, mezo/iso sonrası belirginleşmiş olabilir. Açık tende sık görülür.

### Tedavi

| Yöntem | Niye işe yarar |
|---|---|
| **SPF 50** her gün, geniş şapka, gölge | En kritik — yeni lekeleri engeller |
| **ISANA Vit C 10%** + **niacinamide** | Mevcut lekeleri açar |
| **Tretinoin** | Hücre yenileme |
| **Azelaic** | Spot bazlı |

### Beklenti

**6+ ay**. Güneş lekeleri en yavaş soğuk. Bazıları lazer/IPL gerektirebilir (gelecek, doktor değerlendirmesi).

## 🌑 4. Periorbital + Perioral Hiperpigmentasyon

Göz kirpik dibi + dudak çevresi koyuluk.

### Sebep

- Hassas + ince cilt
- Kronik sürtünme (sürme, ovalama)
- Atopi/alerjik temas
- Mezo / lazer sonrası olabilir
- Genetik (Akdeniz ten)

### Tedavi

**Dikkatli yaklaşım gerekli — bu bölgelerde tret YASAK**:
- Geceleri parmakla **azelaic 10% çok ince katman** → güvenli
- Kirpik dibine sadece **nemlendirme + SPF**
- UV gözlüğü yardımcı olabilir
- Dudak çevresi için **Bepanthol Lip / kalın petrolatum** koruma
- Sabah ISANA Augencreme 0%; Akşam ISANA Peptid Augencreme

### Beklenti

Aylar/yıllar süren değişim. Kalıcı olabilir. Hedef: koyulaşmayı durdur + hafif açma.

| **Azelaic acid** | ✅✅ ana | ✅✅ ana | ✅ destek | ✅ (parmakla ince) |

| **Centella (Madescar)** | ✅✅ kızarıklık + iyileşme | ✅ | ✅ | ✅ |

## ⏱ Genel Beklenti Yönetimi

- **PIE (kırmızı)**: niacinamide + azelaic + SPF + zaman → **2-3 ay** belirgin azalma
- **PIH (kahve)**: tret + Vit C + azelaic + SPF → **3-6 ay**
- **Güneş lekeleri**: SPF + Vit C + tret → **6+ ay**
- **Periorbital/perioral**: kalıcı olabilir, hedef koyulaşmayı durdur

**İlk 2-3 hafta "purging"** olabilir (Acnelyse) — geçici akne artışı, panik yok, devam.
