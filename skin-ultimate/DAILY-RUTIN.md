# Günlük Cilt Rutini — Sabah + Akşam

## ☀️ Sabah (her gün)

1. **Yıkama** → Sabah → sadece su + havlu kurutma (ya da Garnier Vit C cleanser sadece T-bölge — batıyor/kuruyorsa kullanma)
2. **Hyaluron + Panthenol serum** (ISANA) → nem havuzu
3. **Niacinamide serum** (yeni alınacak) → kuruyana kadar bekle
   - *Alternatif sabah*: **ISANA Vit C 10%** → niacinamide YERİNE haftada 3-4 sabah (basit tutuluyor; aynı seansta her ikisi gerekli değil)
4. **Garnier Hyaluron Fresh** nemlendirici → kurusa Madescar küçük miktar üstüne
5. **ISANA Augencreme 0%** → göz çevresi
6. **Garnier BHA + Niacinamide SPF 50** → 2 parmak boyu, kulağa+boyuna. **Her sabah, mecbur.**
7. *(İsterse)* **Purest Solutions BB cream** SPF üstüne

**Kural**: Vit C ve niacinamide aynı sabah birlikte değil — alternatif günlerde.

## 🌙 Akşam — Default (haftanın 4-5 günü: Çar + Cum bariyer; Pzt + Cmt azelaic)

### Default bariyer akşamı (Çar + Cum pilates sonrası)

1. Makyaj varsa → yumuşak yağ/balm temizleyici **ya da** yumuşak su+temizleyici çift yıkama (ISANA micellar BIRAK)
2. Yumuşak temizleyici (yeni alınan CeraVe/Bioderma) ile tüm yüz
3. **ISANA Hyaluron + Panthenol serum**
4. **Madescar** ince katman → bariyer + centella
5. **ISANA Peptid Augencreme** → göz çevresi
6. *(Çok kurursa)* **Garnier Hyaluron Fresh** ekle

### Azelaic akşamı (Pzt + Cmt)

1. Temizlik (yumuşak)
2. **ISANA Hyaluron + Panthenol serum**
3. **Azelaic acid 10%** (yeni alınacak) — PIE+akne+leke tek vuruşta. Kuruyana kadar bekle.
4. **Madescar** ince katman
5. **ISANA Peptid Augencreme**
6. *(Çok kurursa)* Garnier Hyaluron Fresh

## 🌙 Tretinoin (Acnelyse) Gecesi — haftada 2 (Sal + Paz)

**Sandwich Tekniği** (hassas ciltte irritasyonu azaltır):

1. Temizlik (yumuşak)
2. Cilt **tamamen kurusun** (10 dk)
3. **Madescar ince katman** (sandwich katmanı 1)
4. **Acnelyse** mercimek tanesi büyüklüğünde — tüm yüze yay (göz çevresi + dudak köşeleri hariç)
5. 5-10 dk bekle
6. **Madescar veya Garnier Hyaluron** üstüne (sandwich katmanı 2 — nemlendirici kapama)
7. **Göz**: ISANA Peptid Augencreme

**Kurallar**:
- Tret gecesinde Vit C, AHA, niacinamide bindirme YAPMA
- Tret + yoğun terleme/duş = irritasyon → o akşam SKIP, sıradaki sakin akşama kaydır
- Çar + Cum 18:00 pilates dersi akşamlarında tret YASAK

## 🌙 AHA Gecesi — haftada 1 (Perşembe)

1. Temizlik
2. Cilt kurusun (ağırlık antrenmanı varsa duş sonrası 1 saat bekle)
3. **ISANA AHA 6%** ince katman, 10 dk bekle
4. **Madescar + nemlendirici**

**Kural**: AHA gecesinden sonraki gün tret/azelaic ATLA. Ertesi gün (Cuma) bariyer günü → SPF + nemlendirici. Cuma'da zaten pilates dersi var → bariyer uyumlu ✓

## 👁 Gözaltı Bakımı (her gün)

- **Sabah**: ISANA Augencreme 0% — sade baz
- **Akşam**: ISANA Peptid Augencreme — peptid + ince çizgi
- **Şişlik için**: Gece kafa hafif yüksek (ekstra yastık); sabah soğuk metal kaşık 1-2 dk
- Su tüketimi günde 5+ bardak (zaten protokolde) ✓

## 💋 Dudak Çevresi + Kirpik Dibi Hiperpigmentasyon

- Bu bölgelere **Acnelyse uygulamayın** — irritasyon riski yüksek
- Geceleri parmakla **azelaic 10%** çok ince katman → güvenli
- Kirpik dibine sadece nemlendirme + SPF (UV gözlüğü yardımcı olabilir)
- Dudak çevresi için kalın petrolatum/Bepanthol Lip ekstra koruma

## 🛁 Vücut (sırt + dekolte siyah nokta + nadiren akne)

- Duş sonrası → SA içerikli body lotion (örn. Naturium SA 2% / Some By Mi / Inkey List BHA Body Lotion)
- Spor sonrası **HEMEN** duş + kıyafet değiştir
- Pamuk sutyen/iç çamaşırı + yumuşak deterjan
- Sırt sürtünmeyen omuz çantası
