# Cilt Master Sentez

> **Hazırlayan**: Begümnaz ablası + Claude
> **Tarih**: 12 Mayıs 2026
> **Kaynak**: Plan dosyası Bölüm 2 (2.1-2.12)

## 🎯 Çekirdek Strateji: Bariyer-Önce

Begümnaz'ın cildi **8 ay × 40 mg isotretinoin (Zoretanin, etki etmedi) + 3× akne+scar mezoterapi** geçmişi taşıyor. Sonuç: **post-iso incelmiş bariyer + hassas + dehidre**. Üzerine **hormonal akne (jawline, endo flare ile koreleli) + PIE (kırmızı izler) + PIH (kahve) + güneş lekeleri** karışımı.

**Yaklaşım**: Minimum efektif doz. Aktifleri yığma. Önce 2 hafta bariyer kur, sonra hafta hafta aktif ekle.

## 📊 Cilt Profili

| Özellik | Durum |
|---|---|
| Tip | Karma → bölgesel kuruluk + hassas + dehidre |
| Ton | Açık ten (güneş hasarına meyilli) |
| Aktif sorun | Yanak/çene/jawline akne (hormonal — endo flare) + patlatma alışkanlığı |
| Leke profili | **Baskın**: PIE (kırmızı/pembe izler) → niacinamide+azelaic+SPF+zaman 2-3 ay<br>**Az**: PIH (kahverengi) → tret+VitC+azelaic+SPF 3-6 ay<br>**Güneş lekeleri** + yanaklarda çil benzeri → SPF+VitC+tret 6+ ay |
| Periorbital | Göz kirpik dibi + dudak çevresi koyuluk (perioral hiperpigmentasyon) |
| Diğer | Glow eksikliği, mat görünüm, dehidrasyon çizgileri, gözaltı koyuluk + şişlik |
| Dekolte/sırt | Ara sıra siyah nokta tipi komedonlar |

> **KRİTİK**: Acnelyse 0.025% **retinol değil, tretinoin** (prescription retinoid). Çok daha güçlü. Hassas ciltte 2x/hafta dozu doğru.

## 🎯 Leke Profili Stratejisi

Detay → [`PIE-PIH-GUNES-LEKE-STRATEJI.md`](PIE-PIH-GUNES-LEKE-STRATEJI.md)

- **PIE (kırmızı, baskın)**: niacinamide + azelaic + SPF + zaman → 2-3 ay
- **PIH (kahve)**: tret + Vit C + azelaic + SPF → 3-6 ay
- **Güneş lekeleri**: SPF + Vit C + tret → 6+ ay
- **Aktif akne**: tret + azelaic + (endo hormon kontrolü) → 4-8 hafta
- **İlk 2-3 hafta "purging" olabilir (Acnelyse) → panik yok, devam**

## ⚠️ Kaçınılacaklar

- ❌ **Sıkma/patlatma** (PIE+PIH ana sebebi)
- ❌ ISANA Micellar (yakıyor)
- ❌ Parfümlü ürünler
- ❌ Tret + AHA + Vit C bindirme
- ❌ Tret gündüz, SPF'siz dışarı
- ❌ Çok sıcak su yıkama
- ❌ Yüze tırnakla dokunma alışkanlığı
