# Agent 01 — Dermatolog (Genel + Hassas Cilt + Post-Iso Bariyer)

> Perspektif: Hassas + post-isotretinoin bariyer profili olan hastayı değerlendiren bir genel dermatolog. Bu perspektif SYNTHESIS'in bariyer-önce yaklaşımının dayanağı.

## 🎯 Bu Perspektifin Vurguları

### 1. Post-Isotretinoin Bariyer = Yıllar Süren İncelmiş Cilt

**8 ay × 40 mg Zoretanin** kullanan bir hastada cilt:
- Sebase bezler küçülmüş (yağ üretimi minimum)
- Stratum corneum incelmiş (TEWL artmış)
- Bariyer fonksiyonu kompromize
- UV hassasiyet kalıcı yüksek
- Mukoz membran fragility (dudak, göz çevresi)

**Sonuç**: İlaç 8 ay önce bitse bile cilt kalıcı değişiklik. **Aktif yığma = irritasyon, PIE, bariyer hasarı**.

### 2. Mezoterapi Sonrası 8 Hafta Hassasiyet

3× mezoterapi (en yenisi 2 ay önce — Mart 2026). Mezoterapi sonrası:
- 4-6 hafta cilt damarları sensitize
- 8 hafta tam restore
- Aktif kullanımı bu süreçte risk

**Begümnaz'da**: Mart 2026 → Mayıs 2026 = 8 hafta dolmak üzere. Yeni aktifleri (azelaic, niacinamide) güvenle başlatılabilir.

### 3. Bariyer-Önce Yaklaşımı Bilimsel Kanıt

Sensitiv + post-iso ciltte aktif başlatmadan önce **2 hafta bariyer kuruluşu**:
- Yumuşak temizleyici (non-stripping)
- Hyaluronic acid (humektan)
- Ceramide / panthenol (bariyer recovery)
- Niacinamide bile başlangıçta beklenmeli — sebum + sensitivity için ideal ama bariyer toparlandıktan sonra

### 4. Aktiflerin Sırası (Toleransa Göre)

Hassas cilt için önerilen ekleme sırası:
1. **Bariyer + nemlendirici + SPF** (her zaman)
2. **Niacinamide 5-10%** (en güvenli aktif; sebum + anti-inflam + lekecil)
3. **Azelaic acid 10%** (sıradan irritan değil; comedolytic + lekecil + anti-inflam)
4. **Vitamin C 5-10%** (hassas cilt için L-ascorbic değil, etil askorbat veya tetraheksildecyl ascorbate gibi türev)
5. **Retinoid** (Acnelyse 0.025% tretinoin — burada zaten var, çevre + zaman)
6. **AHA glycolic** (en sona — en irritan)

Begümnaz'ın 8 haftalık planı bu sıraya uyuyor ✓

### 5. Tretinoin Uygulama Optimizasyonu

**Sandwich tekniği** (bizim plana zaten dahil):
- Madescar → tret → Madescar = irritasyon %50 azalır
- Kısa kontak yöntemi (10 dk uygula → yıka) toleransa göre opsiyon

**Uygulanmayan bölgeler**:
- Göz çevresi (perioribital atrofi riski)
- Burun kıvrımları (toplanma → tahriş)
- Dudak köşeleri (perioral dermatit)

### 6. SPF Mecbur (50+, Geniş Spektrum)

Post-iso ciltte SPF olmadan dışarı çıkmak = PIH/güneş leke garantisi.

Garnier BHA + Niacinamide SPF 50 = ideal seçim:
- Mineral + kimyasal kombinasyon
- Niacinamide ek
- BHA blackhead destek (dekolte/sırt için faydalı)

### 7. Bu Perspektifin SYNTHESIS'e Katkısı

- **Bariyer-önce 2 hafta** geçişi
- **Sandwich tekniği** tret protokolünde
- **8 haftalık aşamalı plan**
- **Sıkma/patlatma kesin yasak**
- **Tret + AHA + Vit C bindirme yasağı**
- **Tret bölgesel uyarılar** (göz çevresi, dudak köşeleri)
