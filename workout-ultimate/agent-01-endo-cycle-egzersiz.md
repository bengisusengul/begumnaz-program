# Agent 01 — Endometriozis + Cycle-Aware Egzersiz Uzmanı

> Perspektif: Endometriozis tanılı kadına özel egzersiz programı tasarlayan jinekolojik fizyoterapi + endokrin-bilinçli antrenör.

## 🎯 Bu Perspektifin Vurguları

### 1. Endo + Egzersiz İlişkisi

Endometriozis cerrahi sonrası iyileşme + içerde kalan dokunun yan etki kontrolü için egzersiz **kanıtlı fayda**:
- Anti-inflamatuvar etki (orta yoğunluk)
- Pelvik kan dolaşımı destek
- Endorphin → ağrı azaltma
- Stres + kortizol denge (endo flare tetiği)

**AMA aşırı yoğunluk = ters etki**:
- Kortizol artışı → östrogen dominance → endo aktivasyonu
- Pelvik basınç artışı → ağrı tetiği
- RED-S (relative energy deficiency in sport) → menstrüel düzensizlik

**Begümnaz'ın programı** orta yoğunluk ağırlık + pelvik taban + yürüyüş + pilates = ideal denge.

### 2. Menstrüel Döngü = Egzersiz Periyodizasyonu

Modern spor bilimi son 10 yılda kadın atlete cycle-based training önemini doğruladı:

**Foliküler faz (6-12. gün)**:
- Yüksek östrojen → kas büyümesi + recovery hızlı
- **Heavy lift + yüksek volume** = ideal
- En çok kg arttırılabilir

**Ovulasyon (13-15. gün)**:
- Östrojen + testosteron peak
- **PR (personal record)** denenebilir
- AMA endo flare riski → o gün hassas

**Luteal erken (16-22. gün)**:
- Progesteron yükseliyor
- Vücut ısısı + nabız yükselir
- **Volume sürdür**, intensity hafif düşür

**Luteal geç (23-28. gün, PMS)**:
- Hormonlar düşüş + PMS
- Recovery yavaşlar
- **Yoğunluk %20-30 düşür** — pilates + pelvik taban öne

**Menstrüel (1-5. gün)**:
- Hormonlar dipte
- **Aktif iyileşme** (yürüyüş + magic circle + heating pad)
- 3-5. gün'de yumuşak başlayarak normal şemaya geri

### 3. Pelvik Taban Fizyopilates = Hardal

Endo'da **pelvik taban disfonksiyonu** %50+ oran:
- Kronik pelvik ağrı → pelvik taban hipertonisi (sürekli kasılma)
- Cerrahi sonrası adhesion → fonksiyonel bozulma
- Disparoni (cinsel ilişki ağrısı)
- İdrar/bağırsak sorunları

**Pelvik taban fizyopilates** çözüm sağlar:
- Diyaframatik nefes → pelvik taban relax
- Kegel + relax dönüşümü (sadece kasılma değil, gevşeme de)
- Hip mobility integration
- Core stabilization

**Begümnaz programında haftada 2x = ideal**. Aşırı dozaj (3+ haftada) gerekmiyor.

### 4. Ağırlık Antrenmanı + Endo

Ağırlık antrenmanı endo'da **güvenli + faydalı** (kanıtlanmış):
- Kemik yoğunluk koruma
- Sarcopenia önleme
- Cinsel hormon dengeleme
- Insulin sensitivity artırma

**AMA dikkat edilmesi gerekenler**:
- **Çok yüksek intra-abdominal basınç** → pelvik taban zorlanır (max effort 1RM nadiren)
- **Valsava manevrası** (nefes tutarak ağır kaldırma) → tercih edilmez
- **Asla overtraining** → endo flare tetiği
- **Compound liftlere odak** + accessory ile dengelenir

Begümnaz'ın programı moderate weight + higher rep = pelvik taban dostu ✓

### 5. Pilates Dersi (Çar + Cum 18:00)

Begümnaz öğrencisiyle full body pilates yapıyor — bu **çift yönlü fayda**:
- Egzersiz (kendi)
- Öğretmenlik = bilinçli vücut farkındalığı, form kontrol

**Pilates endo için**:
- Düşük impact
- Pelvik taban farkındalığı
- Hip mobility
- Core (transverse abdominis) — superficial six-pack değil, derin core
- Solunum + parasempatik aktivasyon

### 6. Endo Flare Dönemi — Antrenman Modifikasyonu

**Sinyaller**: Bloating + pelvik ağrı + sırt alt kasılma + olağandışı yorgunluk + olağandışı kanama

**Reaksiyon**:
1. **24-48 saat downshift**
2. Ağırlık günü → aktif iyileşmeye çevir (yürüyüş + magic circle iç bacak)
3. Pelvik taban → restoratif (gentle Kegel + breath only)
4. Pilates dersi → öğrencisine "bugün hafif" deyebilir + modifikasyon
5. Heating pad bel/karın 20 dk
6. Ayak banyosu 2x/gün
7. Sıcak su tüketimini arttır

### 7. Bu Perspektifin SYNTHESIS'e Katkısı

- **Cycle-based periyodizasyon** (foliküler push, luteal düşür, menstrüel aktif iyileşme)
- **Pelvik taban haftada 2x** kuralı (yetersiz değil, fazla değil)
- **Ağırlık antrenmanı endo için güvenli** (yüksek volume / moderate weight)
- **Pilates dersi çift yönlü fayda** (öğretmen + katılımcı)
- **Endo flare 24-48 saat downshift kuralı**
