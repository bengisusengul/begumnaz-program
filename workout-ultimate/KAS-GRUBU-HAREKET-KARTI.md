## 🎯 Pratik Kullanım

O günün ortamı belli olur olmaz, ilgili sütundan **4-6 hareket seçilip 30-40 dk'lık bir set kuruluyor**.

**Örnek**: "Bugün ev, glute çalışmak istiyorum"
→ Hip thrust + banded clamshell + magic circle squeeze + side-lying leg lift + donkey kick (büyük top) + glute bridge = 6 hareket → 35 dk

**Örnek**: "Bugün dışarı, alt vücut karışık"
→ Yokuş yürüyüşü 20 dk + yokuşta lunge 3×10 + bench step-up 3×12 + bench glute bridge 3×15 + pull-up bar assisted 3×5 = 45 dk

**Örnek**: "Bugün gym, üst vücut tam"
→ DB shoulder press 3×10 + lat pulldown 3×10 + chest press machine 3×10 + cable row 3×10 + cable curl 3×12 + triceps pushdown 3×12 + face pull 3×15 = 40 dk
